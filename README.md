# VISO App

## Description

VISO est une application mobile qui permet de gérer et de suivre vos tâches efficacement. 

VISO is a mobile app that helps you manage and track your tasks efficiently.

## Features
- Bilingual support (French/English)
- User authentication with email login/signup
- Task management capabilities

## Getting Started

### Prerequisites
- Node.js
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/manimaneya/VISO.git
   cd VISO
   ```
2. Install dependencies for both backend and frontend:
   ```bash
   cd backend
   npm install
   cd ../frontend
   npm install
   ```

### Running the Application

#### Backend

```bash
cd backend
node server.js
```

#### Frontend

```bash
cd frontend
npm start
```
